# 🚖 SmartCab – Console Based Car Rental System

## 🔍 What is SmartCab?

SmartCab is a **console-based car rental application built using Core Java**.

The system allows **Car Owners to list their cars for rent** and **Travellers to search and book available cars**.

This project simulates a **mini ride-sharing / rental platform similar to Uber or Ola** and was built to **strengthen Core Java fundamentals** like Object-Oriented Programming, Collections, Exception Handling, and Multithreading.

---

## 🎯 Project Objective

The goal of this project is to:

• Practice Core Java programming  
• Build a small real-world system using Java  
• Understand Object-Oriented Programming concepts  
• Work with Java Collections Framework  
• Learn Multithreading in a practical way  

---

## 👥 User Roles

SmartCab supports **two types of users**.

### 🚗 Car Owner

Car Owners can:

• Register an account  
• Login to the system  
• Add cars for rent  
• Update car details  
• Delete cars  
• View all listed cars  

---

### 🧑 Traveller

Travellers can:

• Register an account  
• Login to the system  
• View available cars  
• Book cars  
• View booked cars  

---

## 🌟 Key Features

### 🔐 Authentication System

Two separate authentication systems are implemented for:

• Car Owners  
• Travellers  

Users must **register and login before accessing the system**.

---

### 🚗 Car Management (Owner Side)

Car owners can manage their cars using the following operations:

• Add Car  
• Delete Car  
• Update Car Name  
• View All Cars  

Cars are stored using the **ArrayList Collection Framework**.

---

### 🔎 Search Cars

Travellers can view all available cars in the system.

Each car displays:

• Car Name  
• Car Number  

---

### 🚖 Car Booking System

Travellers can book cars by providing:

• Car Name  
• Car Number  

Once the car is booked, the system updates the **car status to "Booked"**.

---

### 📄 View Booked Cars

Travellers can view all cars they have booked.

---

### ⚡ Multithreading Feature

This project uses **Java Multithreading** to simulate a real-time booking notification system.

When a traveller books a car:

• One thread processes the booking  
• Another thread sends a booking confirmation notification  

Example Notification:

Notification: Car Swift booked successfully!
